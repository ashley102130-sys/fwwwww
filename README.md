# 🚀 終極 Discord 炸頻與伺服器管理系統

這是一款功能強大、專為伺服器控制與自動化轟炸設計的進階 Discord 機器人。具備隱身發送、核彈重組與無限語音播放等高阶功能。

---

## 💎 核心功能介紹

### 💣 **毀滅級核彈模式 (`/nuke`)**
終極的破壞與重組工具。一鍵清空伺服器，並按照你的意志重新建立。
- **伺服器重組**：瞬間更改伺服器名稱。
- **大規模平行刪除**：以極速刪除所有現有的頻道與類別。
- **自動重建**：新建立最多 50 個頻道，並立即啟動自動洗頻循環。

### 🎯 **互動式按鈕轟炸 (`/use` & `/dm_spam`)**
極致操作體驗。使用私人的按鈕控制面板，完全繞過 Discord 的發言限制。
- **10 連發點射**：點擊一次按鈕立即發送 10 條訊息。
- **隱身模式**：指令回覆僅限你自己可見（Ephemeral），不留痕跡。
- **全平台支援**：支援使用者安裝模式，即使機器人不在伺服器內也能運作。

### 🔊 **無限語音電台 (`/laugh`)**
讓語音頻道變成永不停止的爆音現場。
- **無限隨機輪播**：自動循環播放 `library` 資料夾中的所有音檔。
- **即時跳過控制**：內建 UI 按鈕，隨時一鍵跳過目前曲目並切換下一首。
- **高效能穩定**：優化的 Windows FFmpeg 串流技術，不噴錯、不卡頓。

### 🌪️ **突擊與快速清理 (`/raid`, `/clear`)**
- **自動突擊**：給予邀請連結，機器人自動加入並對所有頻道展開轟炸。
- **瞬間清理**：一鍵刪除當前頻道最多 100 則訊息。

## 🛠️ 安裝與啟動教學

1. **環境需求**：確保你的電腦已安裝 Python 3.8 或以上版本。
2. **安裝依賴庫**：
   ```powershell
   pip install -r requirements.txt
   ```
3. **設定 Token**：建立一個 `.env` 檔案並填入你的機器人 Token：
   ```env
   DISCORD_TOKEN=你的機器人Token
   ```
4. **一鍵啟動**：
   ```powershell
   .\use.ps1
   ```

---

## ⚙️ 全局管理指令
- **/set_msg**：自訂隨機訊息池，調整所有洗頻功能的發言內容。
- **/stop_all**：緊急停火協議，一鍵停止目前伺服器內所有正在進行的發送任務。
- **/webhook_use**：使用 Webhook 網址進行外部轟炸，不留任何機器人足跡。
